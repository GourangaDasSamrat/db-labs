# Basic Redis Practice

## Prerequisite

- Bun
- Redis uri (your local, docker or any cloud)

## How to use

1. Install dependency
```zsh
bun install
```

2. Setup environment variables
```zsh
cp .env.example .env
```
> And fill with your actual value

3. Run files
```zsh
cd src/labs
```
 And run file with bun.
